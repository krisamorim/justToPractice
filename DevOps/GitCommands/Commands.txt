1- Add al in the stage area
git add .

2- Remove all from stage area 
git reset HEAD -- .

3- Retorna o código para a versão do repositório
revert

4- Ignoar arquivos para não enviar para o repositorio
git ignore


5- Baixar somente atualizações do projeto
pull
obs: se o git pull não funcionar execute o comando git clean -n e rode novamente o pull se não funcionar execute o git clean -d -f e tente novamente
